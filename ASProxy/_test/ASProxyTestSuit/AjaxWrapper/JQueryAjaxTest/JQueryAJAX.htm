Demos » AJAX Plugin

This page demonstrates some quick examples of the new AJAX Plugin for jQuery.

Loaded from an HTML File

Code: 
$("div#html").load("Internal/ajax-test.html");
Loaded from a Text File

Code: 
$.get("Internal/ajax-test.txt",function(txt){
  $("div#txt").html("<h1>"+txt+"</h1>");});
Loaded from a Dynamic HTML File

Code: 
$("div#dhtml").load("/ASProxy/ajax.ashx",{name:"John"});
