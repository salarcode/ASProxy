How to Enable Cookies and Javascript

Both cookies and javascript must be enabled in your browser. If one of these features is disabled, you won't be able to use VenueSystems.com. If either of the tests below fail, please follow the instructions on this page.

How to Enable Cookies (PC)

  1. Internet Explorer 6.x
    Select 'Internet Options...' from the 'Tools' menu. Select the 'Privacy' tab. Click the 'Advanced...' button. Make sure the 'Override automatic cookie handling' check box is checked. Select the 'Accept' option button under 'First Party Cookies'. Select the 'Accept' option button under 'Third Party Cookies'.
  2. Netscape 8.x
    Select 'Options...' from the 'Tools' menu. Click 'Site Controls' in the left menu. Select 'venuesystems.com' from the 'My Settings' list. Select 'I trust this site' under 'Master Settings' or be sure 'Allow Cookies' is checked under 'Web Features'.
  3. Netscape 6.x, 7.x
    Select 'Preferences...' from the 'Edit' menu. Click 'Privacy & Security' in the left window pane under 'Category'. Click 'Cookies' in the expanded menu. Select the 'Enable all cookies' option button.
  4. Firefox 1.x
    Select 'Options...' from the 'Tools' menu. Click 'Privacy' in the left menu. Expand the 'Cookies' section. Check 'allow sites to set cookies' and be sure venuesystems.com is not listed under 'Exceptions'.

How to Enable Cookies (Mac)

  1. Internet Explorer 5.x (OS 9)
    Select 'Preferences...' from the 'Edit' menu. Double-click on 'Receiving Files', then scroll down and click on 'Cookies'. Next to the words 'When receiving cookies' choose 'Never ask'.
  2. Internet Explorer 5.x (OS X)
    Select 'Preferences...' from the 'Explorer' menu. Double-click on 'Receiving Files', then scroll down and click on 'Cookies'. Next to the words 'When receiving cookies' choose 'Never ask'.
  3. Netscape 6.x, 7.x (OS 9)
    Select 'Preferences...' from the 'Edit' menu. Click 'Privacy & Security' in the left window pane under 'Category'. Click 'Cookies' in the expanded menu. Select the 'Enable all cookies' option button.
  4. Netscape 7.x (OS X)
    Select 'Preferences...' from the 'Netscape' menu. Double-click on 'Privacy & security'. Select 'Cookies'. Make sure 'Enable all cookies' is checked.
  5. Safari 1.x
    Select 'Preferences...' from the 'Safari' menu. Click 'Security'. Check 'Accept cookies only from sites you navigate to'.

How to Enable Javascript (PC)

  1. Internet Explorer 6.x
    Select 'Internet Options...' from the 'Tools' menu. Click the 'Security' tab and then click the 'Custom Level' button. Find the 'Scripting' category and click 'Enable' under 'Active Scripting'.
  2. Netscape 8.x
    Select 'Options...' from the 'Tools' menu. Click 'Site Controls' in the left menu. Select 'noodletools.com' from the 'My Settings' list. Select 'I trust this site' under 'Master Settings' or be sure 'Enable Javascript/VBScript' is checked under 'Web Features'.
  3. Netscape 6.x, 7.x
    Select 'Preferences...' from the 'Edit' menu. Click "Advanced" and then click on the "Enable JavaScript for Navigator" check box.
  4. Firefox 1.x
    Select 'Options...' from the 'Tools' menu. Click 'Web Features' in the left menu. Check 'Enable JavaScript'.

How to Enable Javascript (Mac)

  1. Internet Explorer 5.x
    Select 'Preferences...' from the 'Edit' menu. Under 'Web Browser', select 'Web Content'. In the 'Active Content' section, check the 'Enable scripting' checkbox.
  2. Netscape 6.x, 7.x
    Select 'Preferences...' from the 'Edit' menu. Double-click the 'Advanced' option in left panel. Find the 'Enable Javascript for' section on the right panel. Check the checkbox for 'Navigator'
  3. Safari 1.x
    Select 'Preferences...' from the 'Safari' menu. Click 'Security'. Check 'Enable JavaScript'.
